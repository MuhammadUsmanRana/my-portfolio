# Portfolio Website

A modern portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

- Responsive design
- Dark/Light mode
- Contact form
- Social media integration
- Modern UI/UX

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started

1. Clone the repository
```bash
git clone [your-repository-url]
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```
